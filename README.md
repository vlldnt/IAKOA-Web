## Technologies utilisées

- **Backend**  
  - Langage : TypeScript  
  - Framework : Node.js avec NestJS  
  - Points forts : Architecture modulaire, controllers/services, injection de dépendances. Authentification avec bcrypt (hachage des mots de passe) et tokens (JWT).

- **Frontend**  
  - Langage : TypeScript  
  - Framework : React avec Next.js  
  - Points forts : Rendu côté serveur / génération statique, routage et pages. Styles et utilitaires UI avec Tailwind CSS.

- **Base de données**  
  - Technologie : PostgreSQL  
  - Outil : Prisma ORM  
  - Points forts : Schéma typé, migrations, et requêtes type-safe.
